# E-Commerce Executive & Operations Analytics Suite

A multi-page Power BI portfolio project simulating a 
direct-to-consumer e-commerce brand.

## Pages
- Executive Summary
- Sales Analysis (in progress)
- Customer Analytics (in progress)
- Operations (in progress)

## Tech Stack
- Power BI Desktop
- Python (data generation)
- Figma (custom backgrounds)
- DAX (measures library)

## Data Model
Star schema with 2 fact tables and 4 dimension tables
- 6,000 orders | 732 returns | 800 customers | 120 products
